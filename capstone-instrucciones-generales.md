## Step 2: Deploy your project on GitHub Pages
To simplify the review process, create a GitHub repository and build a GitHub Page for your project. The steps for creating the GitHub Page will vary slightly depending on your coding process.

Refer to the GitHub Pages help page for the most up-to-date instructions for creating a page from your project.

The general steps are:

1. Sign in to your GitHub account
2. Create a new repository
3. Upload your HTML and CSS files
4. Commit your changes

   - If this is your first commit for this project, you can use the commit message Initial commit

5. Enable Github Pages by selecting a source

   - From GitHub: Click on the Settings tab and scroll down to the GitHub Pages section. Then select the main branch source and click on the Save button.

6. Preview your page by going to http://username.github.io/repository where username is your username and repository is the name of your project repository
